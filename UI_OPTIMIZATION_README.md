# 智研前端 UI 优化说明

## 优化概述

本次UI优化旨在为智研前端项目创建一套现代化、一致性的设计系统，提升用户体验和视觉效果。

## 主要改进

### 1. 设计系统 (Design System)

#### 颜色系统
- **主色调**: 采用现代化的蓝色系 (#2563eb)
- **辅助色**: 灰色系，提供良好的层次感
- **状态色**: 成功(绿色)、警告(橙色)、错误(红色)、信息(青色)
- **中性色**: 从浅到深的灰色渐变，支持不同层级的内容

#### 字体系统
- **字体族**: 系统字体栈，确保跨平台一致性
- **字体大小**: 从12px到48px的完整尺寸体系
- **字体粗细**: 从100到900的完整权重体系
- **行高**: 针对不同场景优化的行高设置

#### 间距系统
- **基础单位**: 4px的倍数系统
- **间距范围**: 从4px到96px的完整间距体系
- **响应式**: 针对不同屏幕尺寸的间距调整

#### 圆角系统
- **小圆角**: 4px (按钮、标签)
- **中圆角**: 6-8px (卡片、输入框)
- **大圆角**: 12-16px (容器、模态框)
- **全圆角**: 9999px (圆形元素)

#### 阴影系统
- **轻微阴影**: 用于卡片和按钮
- **中等阴影**: 用于悬停效果
- **重阴影**: 用于模态框和下拉菜单
- **特殊阴影**: 用于强调和层次感

## 使用指南

### 颜色使用
```css
/* 主色调 */
color: var(--primary-color);
background: var(--primary-color);

/* 文字颜色 */
color: var(--text-primary);
color: var(--text-secondary);
color: var(--text-tertiary);

/* 背景颜色 */
background: var(--bg-primary);
background: var(--bg-secondary);
```

### 间距使用
```css
/* 使用间距变量 */
padding: var(--space-4);
margin: var(--space-6);
gap: var(--space-3);
```

### 字体使用
```css
/* 字体大小 */
font-size: var(--text-sm);
font-size: var(--text-base);
font-size: var(--text-lg);

/* 字体粗细 */
font-weight: var(--font-medium);
font-weight: var(--font-semibold);
font-weight: var(--font-bold);
```

### 圆角使用
```css
/* 圆角 */
border-radius: var(--radius-sm);
border-radius: var(--radius-md);
border-radius: var(--radius-lg);
```

### 阴影使用
```css
/* 阴影 */
box-shadow: var(--shadow-sm);
box-shadow: var(--shadow-md);
box-shadow: var(--shadow-lg);
```
