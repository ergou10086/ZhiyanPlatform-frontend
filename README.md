# 智研平台前端项目

基于Vue2开发的智研平台前端项目，包含登录、注册和忘记密码功能。

## 功能特性

- 🔐 用户登录页面
- 📝 用户注册页面  
- 🔑 忘记密码页面
- 📱 响应式设计
- 🎨 现代化UI界面

## 技术栈

- Vue 2.6.14
- Vue Router 3.5.4
- Axios 0.27.2

## 项目结构

```
src/
├── views/           # 页面组件
│   ├── Login.vue           # 登录页面
│   ├── Register.vue        # 注册页面
│   └── ForgotPassword.vue  # 忘记密码页面
├── router/          # 路由配置
│   └── index.js
├── App.vue          # 根组件
└── main.js          # 入口文件
```

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run serve
```

3. 访问 http://localhost:8080

## 页面路由

- `/` - 重定向到登录页面
- `/login` - 登录页面
- `/register` - 注册页面
- `/forgot-password` - 忘记密码页面

## 页面功能

### 登录页面
- 用户名/邮箱登录
- 密码输入
- 记住我选项
- 忘记密码链接
- 注册页面跳转

### 注册页面
- 用户名输入
- 邮箱输入
- 密码设置
- 确认密码
- 手机号输入
- 用户协议确认

### 忘记密码页面
- 邮箱输入
- 用户名输入（可选）
- 发送重置邮件
- 邮件发送成功提示
- 重新发送功能

## 样式特性

- 渐变背景设计
- 现代化卡片布局
- 响应式设计
- 平滑过渡动画
- 统一的色彩方案
